# Lesson 14: Modules and Crates

This lesson covers organizing code with modules and crates in Rust.

## Concepts Covered

- Module system with `mod`
- Public and private items with `pub`
- File-based modules
- Using modules with `use`

## Running the Lesson

1. Navigate to the `lesson14` directory:

   ```bash
   cd lesson14
   ```

2. Run the script:
   ```bash
   ./run_lesson14.sh
   ```

## Expected Output

```
Eating at restaurant
Eating in the garden
Eating a vegetable
```

## Key Takeaways

- Modules organize code into logical units
- `pub` makes items accessible from outside the module
- Files can define modules
- `use` brings items into scope
